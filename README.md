# CodeAlpha AI Internship Tasks

This repository contains AI internship projects completed as part of the CodeAlpha program. The projects demonstrate basic Artificial Intelligence concepts, web development, and machine learning techniques using Python.

---

## 🚀 Project Overview

This repository includes two main tasks:

---

## 🌐 Task 1: Language Translation Tool

A web-based application that translates text between English and Hindi using a Flask-based interface.

### Features:
- User-friendly web interface
- Text translation between English ↔ Hindi
- Input validation
- Copy-to-clipboard feature
- Simple and interactive design

### Technologies Used:
- Python
- Flask
- HTML, CSS
- Requests library

---

## 🤖 Task 2: FAQ Chatbot

An AI-based chatbot that answers user queries using predefined FAQs and NLP techniques.

### Features:
- FAQ-based question answering system
- TF-IDF vectorization
- Cosine similarity for intent matching
- Simple chat interface
- Accurate response selection from dataset

### Technologies Used:
- Python
- Flask
- Scikit-learn
- HTML, CSS

---

## ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/Likhithak309/CodeAlpha.git

2. Navigate to project folder:
cd CodeAlpha

3. Install dependencies:
pip install flask requests scikit-learn

---

## ▶️ Running the Projects

Language Translation Tool:
cd language-translation-tool
python app.py

FAQ Chatbot:
cd faq-chatbot
python app.py
